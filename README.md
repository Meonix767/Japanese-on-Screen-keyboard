# On-Screen Japanese Kana Keyboard

An on-screen Japanese kana keyboard that visually mirrors physical keyboard input to help users learn kana layouts while typing on a standard English keyboard.

## Project Overview

This project explores the idea of combining typing practice with visual feedback for Japanese learners. When active, the application displays an on-screen keyboard arranged in a Japanese kana layout. As the user presses keys on their physical keyboard, the corresponding on-screen key highlights in real time.

The goal is to make it easier for learners to associate physical key positions with Japanese kana, even when using a regular QWERTY keyboard.
